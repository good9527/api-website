# 油价图片API使用说明

## 功能概述

已成功实现从国家发改委官网爬取油价信息并返回图片的功能。

## 已实现的功能

### 1. 数据采集器
- **文件**: `collectors/oil_price.py`
- **功能**: 从发改委官网获取最新油价公告，提取图片附件并下载到本地

### 2. API端点
- **获取最新图片**: `GET /api/v1/oil-price/image`
  - 返回最新的油价图片（JPEG/PNG格式）
  - 自动查找最新的图片文件并返回
  
- **触发数据采集**: `POST /api/v1/oil-price/collect`
  - 手动触发油价数据采集
  - 返回采集结果和状态

### 3. 配置
- **配置文件**: `config/sources.yaml`
- **数据源**: 国家发改委新闻发布列表页面
- **采集频率**: 每天9点（可通过配置修改）

## 使用方法

### 1. 获取最新油价图片
```bash
# 使用浏览器访问
http://localhost:5000/api/v1/oil-price/image

# 使用curl
curl -o oil_price.jpg http://localhost:5000/api/v1/oil-price/image
```

### 2. 触发数据采集
```bash
# 使用curl触发采集
curl -X POST http://localhost:5000/api/v1/oil-price/collect
```

### 3. 启动服务器
```bash
# 启动Flask服务器
python app.py

# 或者使用启动脚本
python start_server.py
```

## 技术细节

### 采集流程
1. 访问发改委新闻发布列表页面
2. 解析HTML，查找包含"成品油价格"的链接
3. 按日期排序，获取最新的公告链接
4. 访问公告页面，提取图片附件
5. 下载图片到本地存储
6. 返回图片路径和API端点

### 图片提取逻辑
- 优先查找附件区域（`div.attachment_l`）中的图片
- 查找包含`W0`、`price`、`oil`等关键词的图片
- 跳过网站图标、logo等小图片

### 错误处理
- 网络请求超时重试（默认3次）
- 图片下载失败时保留旧数据
- 详细的日志记录

## 注意事项

1. **频率限制**: 请避免频繁请求，遵守网站规则
2. **缓存机制**: 图片会缓存到本地，避免重复下载
3. **日志记录**: 采集过程会记录到日志文件
4. **错误处理**: 采集失败时会保留旧数据
